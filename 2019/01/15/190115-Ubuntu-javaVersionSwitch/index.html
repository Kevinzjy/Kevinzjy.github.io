Ubuntu实现Java版本切换

发表于 2019-01-15 | 分类于 Ubuntu |

在使用IGV的时候，发现软件提示需要 java8，然而系统上已经同时安装了 java8 和 java11

可以看到 /usr/bin/java 指向的是 java11 的版本，因此我们只需要重新设置成 java8 就好了

将系统默认使用的 java 版本更改为 java8

输入对应的编号之后，就可以成功使用需要的 jdk 版本了
